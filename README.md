# React API Concepts Demo

A comprehensive React application demonstrating various API integration concepts.

## Features

1. **Reddit API Integration**
   - Fetch posts from any subreddit
   - Real-time search and filtering
   - Infinite scroll
   - Sort by different criteria (hot, new, top)
   - Responsive design

2. **JSONPlaceholder API**
   - CRUD operations simulation
   - Nested data fetching
   - Error handling
   - Loading states

3. **Weather API**
   - OpenWeatherMap integration
   - Form handling with API
   - Error states
   - Loading indicators

## API Concepts Demonstrated

- Axios vs Fetch API
- useEffect for data fetching
- useState for state management
- Custom hooks for API logic
- Error handling strategies
- Loading states
- Debouncing search
- Infinite scroll
- Responsive design
- Component composition
